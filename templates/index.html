Memorial Document Generator API

About this Service

This API processes JSON webhook data to generate personalized memorial documents using Google's Gemini AI. The generated documents are stored in Google Cloud Storage for later retrieval and use.

API Usage

Endpoint: POST /api/process

Request Format

Send a POST request with JSON data in the following format:

{
  "user_id": "unique_user_identifier",
  "reflections": [
    {
      "section": "Life Overview",
      "question": "How would you describe your life's journey?",
      "answer": "I've lived a full life with many adventures..."
    },
    {
      "section": "Childhood and Family Life",
      "question": "What are your fondest childhood memories?",
      "answer": "Growing up on the farm..."
    },
    ...
  ]
}

Response Format

The API will respond with a JSON object indicating success or failure:

{
  "status": "success",
  "message": "Document generated and stored successfully",
  "user_id": "unique_user_identifier",
  "document_url": "gs://bucket-name/user_id/memorial_document_20230615_123456.md"
}

Error Handling

In case of errors, the API will respond with an appropriate error message:

{
  "error": "Error description"
}
